Detailed Description

T.D.A. Ingrediente.

Una instancia c del tipo de datos abstractos Ingrediente es un objeto que contiene la información de un ingrediente, como puede ser su nombre, calorías, Hidratos de carbono, proteínas, grasas, fibra y tipo.
